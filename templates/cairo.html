Samsung Activities

In addition to the upstream proposals mentioned above, the SRA-Silicon Valley team has several cairo-related projects and branches under development:

  • cairogles branches include optimizations for the MSAA backend, and current work on improvements to filters (e.g. gaussian blur, shadow).
  • cairo-nsgl - A cairo branch to enable it to use Apple's NSOpenGL library instead of GLX, thereby removing the need for X11 on OSX.
  • gpu-proxy - Not actually part of Cairo, but provides a virtual OpenGL ES 2.0 driver with state caching and non-blocking EGL operations to enable GLES using applications to run faster.
  • cairo-xcb - Optimizations to cairo when using the XCB libx11.
